# DragoCronometro
Un’applicazione Flutter che combina un cronometro basato su Stream con una simpatica animazione:
un drago che vola attraverso lo schermo mentre il tempo scorre.

## Il progetto utilizza:

StreamController per aggiornare il cronometro in tempo reale

Timer periodici per gestire sia il conteggio del tempo sia l’animazione del drago

Widget Stack per sovrapporre sfondo, cronometro e animazione

Sprite animation tramite sequenza di immagini
